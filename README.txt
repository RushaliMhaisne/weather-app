# Weather App 🌦️

A simple Python app that shows real-time weather info using the OpenWeather API.

## 📋 Features:
- Shows current temperature (°C)
- Describes weather condition (e.g., clouds, rain)
- Shows humidity percentage
- Handles invalid city names
- Shows error if city name is blank

## 🛠️ How to Use:
1. Install the `requests` library in Pydroid 3 (using Pip)
2. Get your free API key from https://openweathermap.org/api
3. Replace the API key in the code:
   API_KEY = "your_api_key_here"
4. Run the script and enter a valid city (e.g., Mumbai, Delhi)

## 🔗 Technologies Used:
- Python 3
- OpenWeatherMap API
- requests library
- JSON data handling
